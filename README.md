# 📉 Customer Churn Prediction

A machine learning web application that predicts whether a customer is likely to churn based on their data. Built with Python and deployed using Docker on AWS EC2.

## 🌐 Live Demo

👉 **[http://customerchurnpredicton.duckdns.org](http://customerchurnpredicton.duckdns.org)**

---

## 🚀 Features

- 🔍 Predict customer churn using ML models
- 📊 Interactive web interface for real-time predictions
- 🐳 Dockerized for easy deployment
- ☁️ Deployed on AWS EC2

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| **Frontend** | HTML, CSS, JavaScript |
| **Backend** | Python (Flask / FastAPI) |
| **ML Model** | Scikit-learn |
| **Containerization** | Docker |
| **Deployment** | AWS EC2 |
| **Domain** | DuckDNS (Free) |

## 📦 Run Locally

### Prerequisites
- Docker & Docker Compose installed

### Steps

```bash
# Clone the repository
git clone https://github.com/kavyaparnami23/customer-churn-predection.git
cd customer-churn-predection

# Run with Docker Compose
docker-compose up --build
```

Then open: [http://localhost](http://localhost)
